# البدء السريع - Quick Start Guide

## للمستخدمين الجدد

### التثبيت السريع (5 دقائق)

#### على Linux/Mac
```bash
# 1. استنساخ المشروع
git clone https://github.com/almashooq1/-.git
cd -

# 2. تشغيل سكريبت الإعداد
./setup.sh

# 3. تشغيل التطبيق
python run.py
```

#### على Windows
```cmd
REM 1. استنساخ المشروع
git clone https://github.com/almashooq1/-.git
cd -

REM 2. تشغيل سكريبت الإعداد
setup.bat

REM 3. تشغيل التطبيق
python run.py
```

### الوصول للنظام
بعد التشغيل، افتح المتصفح على:
```
http://localhost:5000
```

### بيانات الدخول الافتراضية
```
البريد الإلكتروني: admin@alawael.com
كلمة المرور: Admin@123
```

**⚠️ تحذير مهم**: غيّر كلمة المرور فوراً بعد أول تسجيل دخول!

## للمطورين

### إعداد بيئة التطوير

#### 1. المتطلبات
- Python 3.8+
- pip
- Git

#### 2. الإعداد
```bash
# استنساخ المشروع
git clone https://github.com/almashooq1/-.git
cd -

# إنشاء البيئة الافتراضية
python -m venv venv

# تفعيل البيئة الافتراضية
# على Linux/Mac:
source venv/bin/activate
# على Windows:
venv\Scripts\activate

# تثبيت المتطلبات
pip install -r requirements.txt

# نسخ ملف البيئة
cp .env.example .env
# قم بتحرير .env وتعيين القيم المناسبة

# تهيئة قاعدة البيانات
python run.py
```

#### 3. التشغيل
```bash
python run.py
```

## باستخدام Docker

### التشغيل السريع
```bash
# استنساخ المشروع
git clone https://github.com/almashooq1/-.git
cd -

# نسخ ملف البيئة
cp .env.docker .env
# قم بتحرير .env وتعيين القيم المناسبة

# تشغيل التطبيق
docker-compose up -d

# الوصول للتطبيق
# http://localhost:5000
```

### إيقاف التطبيق
```bash
docker-compose down
```

### عرض السجلات
```bash
docker-compose logs -f
```
