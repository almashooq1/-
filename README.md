# مراكز الأوائل للرعاية النهارية
## نظام إدارة شامل ومتكامل

### نظرة عامة
نظام إدارة شامل لمراكز الرعاية النهارية، مصمم لتسهيل عمليات الإدارة اليومية وتتبع تقدم الطلاب وإدارة الموظفين والبرامج العلاجية.

### المميزات الرئيسية
- 👥 **إدارة الطلاب**: تسجيل ومتابعة الطلاب وأولياء الأمور
- 👨‍🏫 **إدارة المعلمين والموظفين**: إدارة الموارد البشرية والحضور والإجازات
- 📚 **إدارة الفصول الدراسية**: تنظيم الفصول وتوزيع الطلاب
- 📊 **تقييم المهارات**: نظام متقدم لتقييم مهارات الطلاب وتتبع تقدمهم
- 🏥 **العيادات المتخصصة**: إدارة المواعيد والجلسات العلاجية
- 📋 **المقاييس النفسية**: تطبيق مجموعة واسعة من المقاييس المعتمدة
- 🤖 **الذكاء الاصطناعي**: توصيات ذكية وتحليلات متقدمة
- 🚨 **إدارة الطوارئ**: نظام متكامل لإدارة الحالات الطارئة
- 📱 **تطبيق السائق**: متابعة النقل والتواصل مع الأهالي

### المتطلبات
- Python 3.8 أو أحدث
- pip (مدير حزم Python)
- SQLite (للتطوير) أو PostgreSQL (للإنتاج)

### التثبيت والإعداد

#### 1. استنساخ المستودع
```bash
git clone https://github.com/almashooq1/-.git
cd -
```

#### 2. إنشاء بيئة افتراضية
```bash
python -m venv venv

# على Windows
venv\Scripts\activate

# على Linux/Mac
source venv/bin/activate
```

#### 3. تثبيت المتطلبات
```bash
pip install -r requirements.txt
```

#### 4. إعداد المتغيرات البيئية
```bash
cp .env.example .env
```
ثم قم بتحرير ملف `.env` وتعيين القيم المناسبة.

#### 5. تهيئة قاعدة البيانات
```bash
python -c "from 'ملف موقع الاوائل 2' import app, db; app.app_context().push(); db.create_all()"
```

#### 6. تشغيل التطبيق
```bash
python "ملف موقع الاوائل 2.py"
```

التطبيق سيعمل على `http://localhost:5000`

### البنية التقنية
- **Backend**: Flask (Python)
- **Database**: SQLAlchemy ORM مع دعم SQLite/PostgreSQL
- **Authentication**: JWT (JSON Web Tokens)
- **Frontend**: HTML, CSS, JavaScript
- **Email**: Flask-Mail
- **File Upload**: Werkzeug
