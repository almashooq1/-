# نظام ERP - مراكز الأوائل للرعاية النهارية 2

## نظرة عامة على المشروع

نظام ERP هو منصة متكاملة لتخطيط موارد المؤسسات (ERP) مصممة خصيصًا لمراكز تأهيل ذوي الاحتياجات الخاصة. يوفر النظام إدارة متكاملة للطلاب، والموظفين، والبرامج، والتقييمات، وكافة الجوانب التشغيلية لمراكز التأهيل.

## الميزات الرئيسية

### 1. نظام إدارة شؤون الطلاب
- تسجيل الطلاب وإدارة ملفاتهم الشاملة
- تقييم المهارات (383 مهارة موزعة على 13 فئة)
- تتبع التقدم والتطور
- الخطط التعليمية الفردية (IEP)
- ملفات الطلاب المتكاملة

### 2. نظام إدارة شؤون الموظفين
- ملفات الموظفين الشاملة
- التحكم في الوصول القائم على الأدوار (RBAC)
- توزيع الموظفين على البرامج
- إدارة الحضور والانصراف
- تقييم الأداء

### 3. نظام البرامج التأهيلية
- إدارة برامج التأهيل المتعددة
- خطط التأهيل الفردية
- الجلسات العلاجية وتتبعها
- أهداف SMART Goals
- منهجية متعددة التخصصات

### 4. نظام التقييم والمتابعة
- تقييمات متعددة المجالات
- تقارير التقدم الآلية
- تحليلات مدعومة بالذكاء الاصطناعي
- الاختبارات الموحدة
- مقاييس الأداء

### 5. تكامل الذكاء الاصطناعي
- مساعد ذكي (Chatbot)
- التحليلات التنبؤية
- التعرف على الأنماط
- توصيات علاجية ذكية
- إنشاء المحتوى التلقائي

### 6. نظام الواقع المعزز والافتراضي (AR/VR)
- بيئات علاجية افتراضية
- علامات الواقع المعزز
- سيناريوهات علاجية تفاعلية
- تحليلات الجلسات

### 7. النظام المالي والمحاسبي
- شجرة الحسابات
- قيود اليومية
- الفواتير والمدفوعات
- إدارة البنوك والمصروفات
- التقارير المالية

### 8. نظام الموارد البشرية
- إدارة الموظفين
- تتبع الحضور
- إدارة الإجازات
- تقييم الأداء

### 9. نظام الاتصالات
- رسائل متعددة القنوات (SMS, Email, WhatsApp)

- مكالمات صوتية
- بوابة أولياء الأمور

### 10. تطبيقات الجوال

- تطبيق ولي الأمر
- تطبيق الموظف

## البنية التقنية

### معمارية الخدمات المصغرة (Microservices)
- **API Gateway**: نقطة دخول موحدة
- **خدمات مستقلة**: كل نظام فرعي كخدمة منفصلة
- **Message Broker**: RabbitMQ/Kafka للاتصال غير المتزامن
- **قواعد بيانات متعددة**: PostgreSQL, MongoDB, Redis

### التقنيات المستخدمة
#### Backend
- Python (Flask/FastAPI) - الخدمات الرئيسية والذكاء الاصطناعي
- Node.js (Express/NestJS) - الاتصالات اللحظية
- Go - الخدمات عالية الأداء

#### Frontend
- React.js/Vue.js/Angular - تطبيقات الويب
- React Native - تطبيقات الجوال
- Material-UI / Bootstrap - واجهات المستخدم

#### Database
- PostgreSQL - البيانات المنظمة
- MongoDB - البيانات غير المهيكلة
- Redis - التخزين المؤقت

#### DevOps
- Docker - الحاويات
- Kubernetes - إدارة الحاويات
- Jenkins/GitLab CI - CI/CD
- Prometheus/Grafana - المراقبة
