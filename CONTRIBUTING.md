## 🧪 الاختبارات

### كتابة الاختبارات
```python
import pytest
from app.services.student_service import StudentService

def test_create_student():
    """اختبار إنشاء طالب جديد"""
    service = StudentService()
    student_data = {
        'name': 'محمد أحمد',
        'national_id': '1234567890'
    }
    
    student = service.create(student_data)
    
    assert student.id is not None
    assert student.name == 'محمد أحمد'
    assert student.national_id == '1234567890'

def test_create_student_duplicate_id():
    """اختبار رفض رقم هوية مكرر"""
    service = StudentService()
    
    with pytest.raises(ValueError):
        service.create({'national_id': '1234567890'})
```

### تشغيل الاختبارات
```bash
# جميع الاختبارات
pytest

# اختبار محدد
pytest tests/unit/test_students.py

# مع التغطية
pytest --cov=app --cov-report=html
```

## 🐛 الإبلاغ عن المشاكل

### قبل الإبلاغ:
1. تأكد أن المشكلة لم تُبلغ مسبقاً
2. جرب أحدث إصدار
3. اجمع معلومات التشخيص

### قالب الإبلاغ:
```markdown
## وصف المشكلة
وصف واضح ومختصر للمشكلة.

## خطوات إعادة الإنتاج
1. اذهب إلى '...'
2. انقر على '...'
3. انظر الخطأ

## السلوك المتوقع
ماذا كنت تتوقع أن يحدث؟

## السلوك الفعلي
ماذا حدث فعلياً؟

## لقطات الشاشة
إن وجدت، أضف لقطات شاشة.

## البيئة:
- النظام: [مثلاً Ubuntu 22.04]
- المتصفح: [مثلاً Chrome 120]
- إصدار Python: [مثلاً 3.10]
- إصدار النظام: [مثلاً v1.0.0]

## معلومات إضافية
أي معلومات أخرى مفيدة.
```
